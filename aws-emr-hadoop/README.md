# Laboratório AWS EMR Hadoop

Esta pasta foi criada para armazenar os resultados do laboratório de execução do WordCount com Hadoop MapReduce no Amazon EMR.

## Conteúdo previsto

- Arquivos de saída do processamento WordCount;
- Evidências dos steps executados no Amazon EMR;
- Registros ou observações sobre ajustes realizados durante a execução;
- Documentação complementar do laboratório, quando aplicável.

## Observação

Não devem ser incluídos nesta pasta arquivos com credenciais, chaves de acesso, tokens, arquivos `.pem`, arquivos `.ppk` ou qualquer informação sensível da AWS Academy/Learner Lab.
